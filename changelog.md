# Changelog

## v1.1.1 - 2026-02-14

### Fixed

- Changed keyboard shortcuts to use alt instead of ctrl to avoid conflicts with browser shortcuts across different OS

## v1.1 - 2026-02-13

### Added

- Added keyboard shortcuts:
  - `ctrl + n`: Fetch new challenge
  - `ctrl + r`: Reset current challenge
  - `ctrl + d`: Toggle debug mode (dev only)
- Added daily fetch to challenge to ensure users automatically get a new challenge each day
